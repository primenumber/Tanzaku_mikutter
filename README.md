Tanzaku_mikutter
================

投稿を短冊にするmikutterプラグイン

## How to use

tanzaku.rbを~/.mikutter/plugin以下に置いてmikutterを起動します。

## Install

````
    $ mkdir -p ~/.mikutter/plugin/; git clone https://github.com/primenumber/Tanzaku_mikutter.git ~/.mikutter/plugin/tanzaku/
````
